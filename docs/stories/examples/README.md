# Story Examples

Reference stories that ship with the harness. They demonstrate story shape and serve as worked examples — **they are not active work on a new project**.

When you start a new project, ignore these files. Real stories live at `docs/stories/epics/EXX-*/US-NNN-*.md`.

## Contents

| File | What it demonstrates |
| --- | --- |
